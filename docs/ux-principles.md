# UX Principles (Reference Only)

This document provides conceptual guidance.
It is not a rulebook.

---

## Gestalt Principles

### Proximity
Elements close together are perceived as related.

### Similarity
Elements that look alike are perceived as related.

### Common Region
Elements inside the same container are grouped.

### Figure/Ground
Users distinguish focus vs background.

### Continuity
Users follow visual flow.

### Closure
Users fill in missing information.

### Common Fate
Elements moving together are related.

## Nielsen’s Heuristics

1. Visibility of system status
2. Match with real world
3. User control and freedom
4. Consistency and standards
5. Error prevention
6. Recognition over recall
7. Flexibility and efficiency
8. Aesthetic and minimalist design
9. Error recovery
10. Help and documentation

## Purpose

Use these principles to:
- justify design decisions
- evaluate UI quality
- improve usability
